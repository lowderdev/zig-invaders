Zig Invaders
-- 

Building a space invaders clone by following the course in this youtube playlist by [Code With Cypert](https://www.youtube.com/@CodeWithCypert):  
https://www.youtube.com/watch?v=eZLTAhlFwlI&list=PLYA3HD4nElQnxWnznih9w0RloSw0hFaYQ&index=1

Run with 
```shell
zig build run
```

This was built with Zig `0.15.2`
